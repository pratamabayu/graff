Timeline

The presentation of a chronological sequence of events that enables a user to understand temporal relationships quickly

Usage

Add timeline class in <ul> tag. The content in <li> tag can be anything. Add active class to current <li> tag.

<ul class="timeline">
    <li></li>
    <li class="active"></li>
    <li></li>
</ul>
