{{< heading text="List" sub-text="Set different list item markers for ordered lists and unordered lists" >}}

## Usage

Add `list-*` class to an `<ul>` or `<ol>` tag.

``` html
<ul class="list-style-square">
    <!-- Your content ->
</ul>

<ol class="list-style-upper-alpha">
    <!-- Your content ->
</ol>
```
