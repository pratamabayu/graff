## Usage

Add `overflow-*` class to block or inline block element.

``` html
<div class="overflow-auto">
    <!-- Your content ->
</div>
```

## Classes

{{<table "table table-divider">}}
| Class | Value |
|----------|----------|
| `overflow-auto` |
| `overflow-hidden` |
| `overflow-scroll` |
| `overflow-visible` |
| `overflow-x-auto` |
| `overflow-y-auto` |
| `overflow-x-hidden` |
| `overflow-y-hidden` |
| `overflow-x-scroll` |
| `overflow-y-scroll` |
| `overflow-x-visible` |
| `overflow-y-visible` |
