## Usage

Add `float-*` class to block or inline block element.

``` html
<div class="float-right">
    <!-- Your content ->
</div>
```

## Classes

{{<table "table table-divider">}}
| Class | Value |
|----------|----------|
| `float-none` |
| `float-left` |
| `float-right` |
| `float-clear-both` |
{{</table>}}

> Add suffix `@s`, `@m`, or `@l` to support breakpoints
