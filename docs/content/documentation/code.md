# Code

### Usage

Write your code inside ```<pre><code>``` tag.

```
<pre>
    <code>
        // Your code ...
    </code>
</pre>
```

### Example

```
namespace MyProject
{
    class Program 
    {
        public static void Main(string[] args)
        {
            System.WriteLine("Hello World");
        }
    }
}
```

``` html
<pre>
    <code>
        namespace MyProject
        {
            class Program 
            {
                public static void Main(string[] args)
                {
                    System.WriteLine("Hello World");
                }
            }
        }
    </code>
</pre>
```
